## RULES

1. **Never boost a post with <1% organic engagement.** If nobody cared for free, nobody will care for money.
2. **Kill within 48 hours** if below threshold. Move budget to what's working.
3. **Log every dollar** in the weekly tracker below.
4. **Phase 3 gate:** CPA must be below booking margin. If it's not, stop paid and fix the funnel before spending more.
5. **The $200 is not a target to hit.** If nothing deserves boosting in a given week, don't spend it. Roll it forward.
