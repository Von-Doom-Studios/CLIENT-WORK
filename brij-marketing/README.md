# brij-marketing

The operating folder for Brij marketing. Self-contained operating manual. Anyone new should be able to open `01-GETTING-STARTED.md`, read it, and understand the system inside 10 minutes.

**Owner:** VD-Marketing
**Client:** Brij (https://brij.fi)
**Last updated:** 2026-05-22
