## RULES

1. **Max 1/week.** Launch weeks: max 2.
2. **One message. One CTA.** No exceptions.
3. **Read it aloud before sending.** If it sounds like a press release, kill it.
4. **Unsubscribe > 2%** = you're sending too much.
5. **Subject lines are specific.** If you could swap in any company name and it still works, rewrite it.
