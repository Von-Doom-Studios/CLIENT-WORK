## Ground rules (non-negotiable)

1. **Senior professional bar.** Every artifact in this folder should be something a founder, an agent, or a new hire could pick up and execute the same day. No memos. No frameworks. If it's not actionable, it doesn't belong here.
2. **GitHub is the documentation system.** Not Notion. Not Google Docs. Not Discord threads. Anything that needs to persist lives here.
3. **One file per concern.** Strategy lives in 02. Calendar lives in 05. Campaigns live in 06. Do not mix. Cross-reference instead.
4. **Every file has a header.** Title, summary, owner, related files, last updated date. No exceptions.
5. **Update on change.** When state changes, the file that owns that state gets updated the same day. Stale files are worse than missing ones.
6. **Voice is dry, technical, confident.** Code, numbers, screenshots over adjectives. Builders trust receipts, not takes. No "excited to announce." No rocket emojis. No em dashes.
7. **No token narrative in marketing.** Until and unless that changes via explicit founder approval.
8. **Founders post manually.** No automation until 4-6 weeks of proven manual cadence.
9. **External outreach goes through Ops.** Marketing identifies and briefs; Ops contacts.
10. **Visuals go through VD-Creative.** Marketing briefs; Creative produces.

## How to operate this folder in 5 steps

1. Open [02-STRATEGY.md](./02-STRATEGY.md). Understand the positioning, audience, targets, and what we don't do.
2. Open [04-ACCOUNTS-AUDIT.md](./04-ACCOUNTS-AUDIT.md). Know the current state of @brij_fi and capture surfaces.
3. Open [05-CALENDAR.md](./05-CALENDAR.md). See what's posting this week and next.
4. Check [06-CAMPAIGNS.md](./06-CAMPAIGNS.md) for any active campaign that overrides calendar baseline.
5. Pull post copy from [07-CONTENT-LIBRARY.md](./07-CONTENT-LIBRARY.md), brief visuals to Creative, hand to founder for posting.

## Weekly cadence

- **Monday:** review last week's performance, lock this week's posts in [05-CALENDAR.md](./05-CALENDAR.md), update [11-ANALYTICS.md](./11-ANALYTICS.md).
- **Wednesday:** mid-week engagement push using [08-ENGAGEMENT.md](./08-ENGAGEMENT.md).
- **Friday:** draft next week's posts in [07-CONTENT-LIBRARY.md](./07-CONTENT-LIBRARY.md), brief Creative on any visuals needed.
- **Monthly:** retro - what hit, what flopped, what's the next month's focus. Logged in [11-ANALYTICS.md](./11-ANALYTICS.md).
