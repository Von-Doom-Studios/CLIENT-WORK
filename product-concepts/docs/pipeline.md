# Pipeline Overview

See README.md for full documentation.

## Quick Reference

1. Drop reference image in `products/[name]/reference-images/`
2. Fill out `specs/measurements-template.md`
3. Run `scripts/01_blender_cleanup.py`
4. Run `scripts/02_freecad_to_step.py`
5. Run `scripts/03_generate_spec_sheet.py`
6. Review output in `products/[name]/factory-package/`
7. Order 3D print for prototype validation
8. Ship factory package to Shenzhen
